Archive Of Abject Terror - Hobby Themed Website Prototype

2024 University Project. Prototype website focused on a hobby (Horror media), coded from scratch.

CSS | HTML | JavaScript | Web Design | Without Bootstrap

A first year project, the goal of this project was to create several pages of a website themed around a particular hobby or interest. I chose horror media as I am an avid fan of the genre, and decided to frame my site as a 'database'/review site of sorts.

I coded this prototype mainly with HTML and CSS, and used media queries to make it responsive for a few different screen sizes. Certain interactive elements needed to be included as part of the brief, such as videos, a working form and an image carousel. I was able to implement these features and others with Javascript. The site was coded from scratch without using a framework such as Bootstrap.

Site coded by and all work present developed primarily or in part by Joseph Ansell. 2023-25.
