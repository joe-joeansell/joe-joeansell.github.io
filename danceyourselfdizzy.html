Dance Yourself Dizzy! - Interactive Camera-based P5.JS Game

2025 University Project. P5.JS Based dancing game that uses Machine Learning classifiers and pose tracking to recognise dance moves.

P5.JS | JavaScript | Web/Game Design | Pose Tracking | Classifiers

In this project, I developed on my previous work with p5.js to create a different type of interactive project. The goal for this one was to incorporate a strong machine learning element. My previous major p5.js project had incorporated Markov chains, but this project aimed to use Classifiers and Pose-Tracking to recognise poses within a multiplayer dancing game.

The project utilised the 'BodyPose' model from ml5.js, a model trained to estimate full-body poses. Using this, I trained it on a selection of dance themed poses. The game itself functions as a multiplayer game, with players taking turns to strike poses and scoring points based on whether or not they can accurately match the poses. I also created all of the game's visuals and music myself.
