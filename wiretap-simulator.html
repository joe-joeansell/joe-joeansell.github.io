Wiretap Simulator - Interactive P5.JS Game

2024 University Project. P5.JS Game themed around 'Dystopia'. Browser game inspired by 'Papers, Please'.

P5.JS | JavaScript | Markov Chains | CSS | Graphic Design | Web/Game Design

With this project, the goal was to create an interactive project in the JavaScript library p5.js. It had to reflect some kind of theme, and I selected the theme of 'Dystopia', framing the player as an officer in a dystopian state, tracking down signals and monitoring their contents for dissent.

My project materialised as a game, stylistically inspired by media like "Half Life 2" and "1984". Mechanically, it is similar to the game "Papers, Please", as the player is free to dole judgement on the characters as they see fit. The player's choices ultimately affect the ending they get.

The project was coded entirely in p5.js with some use of CSS, and all of the visuals were created by me, with some elements created in Photoshop and others natively in p5.js.
