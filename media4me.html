Media4Me - Figma Website Prototype

2024 University Project. Figma prototype for a streaming guide website, with rich media elements.

Figma | Photoshop | Web Design | Graphic Design

For this project, the brief was to create a prototype web design, with interactive and rich media features, such as marquee banners, image carousels and animations. The brief was up to me, and I chose to create a streaming guide for TV, Film and Videogames.

I chose to create my prototype in Figma, using this software to design wireframes, as well as final working prototypes for three different screen sizes. This was a good opportunity to develop my web design skills.
